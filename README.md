# 🏢 Management Asset Laravel

Welcome to the **Management Asset Laravel** project! This application helps manage and track assets effectively using the Laravel framework.

## 🚀 Getting Started

### Prerequisites

- PHP >= 7.3
- Composer
- Laravel >= 8.x

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/msaaa26/MANAGEMENT-ASSET-LARAVEL.git
   ```

2. Navigate to the project directory:
   ```bash
   cd MANAGEMENT-ASSET-LARAVEL
   ```

3. Install dependencies:
   ```bash
   composer install
   ```

4. Set up the environment:
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```

5. Run migrations:
   ```bash
   php artisan migrate
   ```

6. Start the server:
   ```bash
   php artisan serve
   ```

## 📚 Usage

After setting up the application, you can access it via `http://localhost:8000` in your web browser.

### Features

- Asset tracking
- User management
- Reports generation
